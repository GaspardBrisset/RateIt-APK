# RateIt APK

Application Android RateIt — télécharge et installe directement sur ton téléphone.

## Installation

1. Télécharge `RateIt-v1.0.apk`
2. Ouvre le fichier sur ton Android
3. Accepte l'installation depuis des sources inconnues si demandé
4. C'est prêt !

## Version

- **v1.0** — version initiale
